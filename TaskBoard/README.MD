Выполнено Агзамовым Артуром.

Проект taskboard реализован при помощи Qt.

Возможности:
 - Создавать проекты(При помощи нажатия соответствующей кнопки)
 - Создавать задачи в проекте при помощи двойного ЛКМ на оси проекта.
 - Перемещать границы задачи.
 - Переименовывать и удалять проекты и задачи при помощи меню, вызываемого ПКМ на обьекте.

В проекте используется компилятор mingw

В планах добавление календаря и возможности создания глобального проекта и задачи(Между днями), а также добавить сохранение сцены в файл(?)
